# Personal Website Starter

## Run locally
From this folder:

```bash
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000).

## Customize first
- `index.html`
- Replace the intro paragraph and project descriptions with your real work.
- Update contact links and email.
- Add your exact school, expected graduation year, and research goals.

## Optional next upgrades
- Add a resume PDF and link it from the hero section.
- Add a publications/presentations section.
- Deploy on GitHub Pages or Netlify.
